## **Menggunakan Phi-4-mini-mm untuk menjana kod**

Phi-4-mini meneruskan keupayaan pengekodan yang kuat daripada Keluarga Phi. Anda boleh menggunakan Prompt untuk bertanya soalan berkaitan dengan pengekodan. Sudah tentu, selepas menambah keupayaan penaakulan yang kuat, ia mempunyai keupayaan pengekodan yang lebih hebat, seperti menjana projek berdasarkan keperluan. Sebagai contoh, menjana projek berdasarkan keperluan, seperti:

### **Keperluan**

Cipta Aplikasi Troli Membeli-belah

- Cipta API Rest dengan kaedah berikut:
    - Dapatkan senarai bir menggunakan offset halaman dan had.
    - Dapatkan butiran bir mengikut id.
    - Cari bir berdasarkan nama, deskripsi, tagline, pasangan makanan, dan harga.
- Cipta senarai produk di halaman utama.
    - Cipta bar carian untuk menapis produk.
    - Navigasi ke halaman deskripsi apabila pengguna mengklik pada produk.
- (Pilihan) Slicer untuk menapis produk mengikut harga.
- Cipta troli membeli-belah.
    - Tambah produk ke dalam troli.
    - Keluarkan produk dari troli.
    - Kira jumlah harga produk dalam troli.
